About the System

A centralized, web-based solution for branch-to-branch deliveries inside Cambodia, designed to give customers, drivers and branch staff a single source of truth for shipment status.

Branch-to-Branch Focus

Designed for shipments moving between provincial branches and cities. Every shipment is attached to origin and destination branches so operations remain clear and auditable.

Clear Roles & Responsibilities

Staff create and manage shipments, drivers update progress, and customers simply enter a tracking code to see where their package is right now.

üìç

Status-Based Tracking

Shipments move through clear statuses such as At Origin, In Transit, Arrived and Delivered.

Optional GPS Location

Drivers can optionally share GPS coordinates so customers can see last-known location during transit.

This project is built as a bachelor degree final project, focusing on realistic workflows used by local logistics providers in Cambodia.
